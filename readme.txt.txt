* replit.com에서 제작 했습니다. 따라서 replit.com에서 실행 해주시는 것이 가장 평가하기에 적절합니다.

* '띄어쓰기로 구분' 이라고 적혀있는 입력들은 필히 따라주시길 권장드립니다.

* 사이트 가입은 어떤 방식으로든 상관없이 해주시면 됩니다.

* 사이트의 왼쪽 상단의 'Create Repl'을 클릭합니다.

* Template에서 Python을 검색 후 클릭, Title은 기본값으로 하셔도 무방합니다.

* 화면의 왼쪽에 'Packger files' 리스트들을 제외한 'Files' 리스트 내의 모든 파일들을 삭제합니다.
(파일 아이콘 위로 마우스 커서를 올리면 우측에 세로 점 3개가 생성됩니다. 클릭 후 'Delete'를 눌러 삭제합니다.)

* Files 옆의 세로 점 3개를 눌러 'Upload file'을 클릭합니다.

* 'main.py' 파일을 클릭해 replit에 업로드 합니다.

* Ctrl + Enter 또는 화면 상단 중앙에 위치한 'Run'을 눌러 실행합니다.
